## 🎁 Features

### ⏰ **Smart Countdown System**
- 🕐 **Real-time Countdown** - Live timer displaying days, hours, minutes, and seconds until Christmas
- 🔄 **Automatic Year Rollover** - Seamlessly transitions to next year's countdown on December 26th
- 📅 **Custom Year Selection** - Choose any year from current year to 35 years in the future
- 📊 **Progress Circle** - Beautiful circular progress indicator showing the journey to Christmas

### 🎨 **9 Stunning Themes**
Each theme features unique gradient backgrounds, color schemes, and visual effects:

| Theme | Description | Colors |
|-------|-------------|--------|
| 🎅 **Classic** | Traditional Christmas | Red & Green |
| ❄️ **Winter** | Icy winter wonderland | Blue & White |
| 👑 **Royal** | Majestic elegance | Purple & Gold |
| 🍭 **Candy** | Sweet and playful | Pink & Red |
| ✨ **Golden** | Luxurious shimmer | Amber & Gold |
| 🧊 **Icy** | Frozen beauty | Slate & Sky |
| 🌲 **Forest** | Deep woodland | Emerald & Teal |
| 🌅 **Sunset** | Warm twilight glow | Orange & Rose |
| 🌌 **Aurora** | Northern lights magic | Indigo & Fuchsia |

### 🎮 **6 Interactive Mini-Games**
Games unlock progressively as Christmas approaches!

1. 🃏 **Memory GIF Game** (25 days) - Match pairs of festive Christmas emojis
2. ❄️ **Catch the Snowflakes** (20 days) - Click falling snowflakes to score points
3. 🎄 **Decorate the Tree** (15 days) - Drag and drop ornaments onto your Christmas tree
4. ⛄ **Build a Snowman** (12 days) - Create your own snowman with various parts
5. 🛷 **Santa Sleigh Ride** (10 days) - Help Santa navigate through obstacles
6. 🍬 **Candy Cane Stack** (5 days) - Stack candy canes as high as possible

### 🌍 **Full Internationalization**
Complete multilingual support in 4 languages:
- 🇬🇧 **English**
- 🇪🇸 **Español** (Spanish)
- 🇫🇷 **Français** (French)
- 🇳🇱 **Nederlands** (Dutch)

*All UI elements, games, themes, and settings are fully translated!*

### 🎬 **Magical Visual Effects**
- ❄️ **Dynamic Snowfall** - Adjustable snow intensity (0-100%) with theme-based colors
- ✨ **Sparkle Particles** - Floating magical particles throughout the screen
- 🎅 **Flying Santa Sleigh** - Animated Santa crossing the winter sky
- 🎊 **Confetti Explosion** - Celebrates when Christmas Day arrives!
- 🎄 **Christmas Eve Tree** - Special animated tree on December 24th
- 🎁 **Animated Gift Boxes** - Festive presents with delightful hover effects

### 🎵 **Music & Audio**
- 🎶 **Background Music Player** - Festive holiday tunes to set the mood
- 🔊 **Volume Control** - Adjustable volume (0-100%)
- ▶️ **Play/Pause** - Full playback controls
- 🔇 **Mute Toggle** - Quick mute option
- 🎼 **Custom Music URL** - Upload your own music via JukeHost.co.uk

### 📱 **Phone-Optimized Mode**
Intelligent mobile detection with optimized UI:
- 📲 **Responsive Design** - Adapts to all screen sizes
- 🎯 **Compact Layout** - Smaller text and spacing for mobile devices
- ⚡ **Performance Optimized** - Removes heavy decorative elements on phones
- 🎮 **Games Hidden on Mobile** - Streamlined experience for small screens
- 👆 **Touch-Friendly** - All interactions optimized for touch screens

### 🔔 **Smart Notifications**
- 📬 **Push Notifications** - Browser notifications for important milestones
- ⏰ **Customizable Alerts** - Set reminders for 1 week, 3 days, and 1 day before Christmas
- ✍️ **Custom Messages** - Write your own notification text
- 🔕 **Easy Toggle** - Enable/disable with a single click

### 💬 **Holiday Inspiration**
- 📖 **Rotating Quotes** - Inspirational holiday quotes that refresh periodically
- 🎄 **Festive Messages** - Heartwarming seasonal greetings

### 🔗 **Social Features**

## 🚀 Getting Started

### **Running the Application**

The application runs automatically in the Leap environment. No installation required!

Simply open the preview URL to experience the Christmas magic. ✨

### **Using the App**

1. **🎯 Select Your Year** - Choose which Christmas you're counting down to
2. **🎨 Pick a Theme** - Select from 9 beautiful color schemes
3. **🎵 Adjust Settings** - Control snow, music, and notifications
4. **🌍 Choose Language** - Switch between 4 supported languages
5. **🎮 Play Games** - Unlock and enjoy festive mini-games
6. **📤 Share Joy** - Share your countdown with loved ones

---

## ⚙️ Settings & Customization

### **Accessible Features**

Click the **menu button** (☰) in the top-right corner to access:

| Setting | Options | Description |
|---------|---------|-------------|
| 🗓️ **Year** | Current to +35 years | Select target Christmas year |
| 🎨 **Theme** | 9 color themes | Change visual appearance |
| ❄️ **Snow Intensity** | 0-100% | Adjust snowfall amount |
| 🔊 **Music Volume** | 0-100% | Control background music |
| 🎼 **Custom Music** | URL input | Use your own JukeHost link |
| 🔔 **Notifications** | On/Off + Custom messages | Configure push alerts |
| 🌍 **Language** | 4 languages | Switch interface language |
| 🎮 **Games** | Hub access | Open mini-games collection |

## 📱 Mobile Experience

The app intelligently detects mobile devices and provides:

- 📐 **Compact Interface** - Optimized spacing and font sizes
- ⚡ **Better Performance** - Removes heavy visual effects
- 🎯 **Focused Content** - Shows only essential countdown elements
- 👆 **Touch-Optimized** - All controls work perfectly on touch screens

**Detection includes:**
- Device user agent (iPhone, iPad, Android)
- Screen width (< 768px)
- Automatic responsive adjustments
